  • "<" é a abertura da tag.
  • "< /" é a fechadura da tag.
  • "href" é o atributo.
  • "google" é aonde fica o conteúdo.
  • "https://google.com" é o valor.

Tag

As tags servem para inserirmos conteúdos como textos, imagens, vídeos e outros. Além da dar informações para o browser como o título do site, linguagem e outras.

Atributo

Os atributos dão informações extras sobre uma tag para o browser.

Tags e atributos com letras maiúsculas ou menúsculas não faz diferença. Porém o valor sim, se for maiúsculo ou menúsculo há valroes diferentes.
