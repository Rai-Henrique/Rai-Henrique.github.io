Listas

marca uma lista de itens sem ordem (unordered list). Cada item da lista deve ser marcado com uma li.

ul-lista não ordenada.

Exemplo:

    ul
  • li que está contindo dentro da ul
  • li que está contindo dentro da ul
  • li que está contindo dentro da ul
  • li que está contindo dentro da ul

Lista dentro de lista

    ul
  • li que está continda dentro da ul
      • ul contida dentro da ul anterior
    • li que está contindo dentro da ul que está contida dentro de outra ul

ol-lista ordenada.

  2. Henrique

  4. Da Silva

dl

CSS

As listas não são utilizadas apenas para textos simples, é possível agruparmos uma lista de produtos em uma ul > li. Para isso geralmente removemos os estilos padrões definidos por listas.

list-style

Possui diferentes valores que definem o estilo dos marcadores da lista (disc, square e outros). O list-style define o tipo / imagem / posição (outside / inside).
