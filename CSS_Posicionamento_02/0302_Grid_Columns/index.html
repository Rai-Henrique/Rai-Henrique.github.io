Grid

Com o Display: grid; é possível definirmos linhas e colunas para organizarmos os elementos que estiverem dentro do grid.

Grid-templates columns:

Define o total de colunas e o tamanho de cada uma.
    fr
  • É uma unidade fracionária que terá como objetivo distribuir o espaço restante.
    gap
  • Cria uma distância entre os elementos do grid, tanto na vertical quanto na horizontal.
    Importante para mim ;-).
  • Ativando o Display: grid; no elemento pai os elementos inline(filhos) deixam de ser inline e recebem valor de block, mas poderia usar Display: inline-block nos filhos, porém posteriormente após usar o Display: grid; no elemento pai onde está o elementos inline(que não seria mais um block) aconteceria conflitos.

Para poder mover os elementos dentro de uma caixa ou definir um gap por exemplo, é preciso ativar o Display: grid;

Ir para prática.
