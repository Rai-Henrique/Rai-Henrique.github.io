Top, Right, Bottom, Left.

Propriedades como margin, padding e border permitem definirmos valores diferentes para cada um dos lados da caixa (box model).

    Padding afasta o conteúdo da borda.

    Margin afasta o conteúdo do body.

    Border separa o padding do margin.

Conteúdo Principal Conteúdo Principal
.padding {
padding-top: 20px;
padding-right: 60px;
padding-bottom: 40px;
padding-left: 10px;
}
Ou resumidamente:
padding{
padding: 20px 60px 40px 10px;
}


.margin {
margin-top: 20px;
margin-bottom: 20px;
margin-right: 40px;
margin-left: 40px;
}
Ou resumidamente:
margin{
margin: 20px 40px;
}

.border {
border: solid black;
border-width: 5px 20px;
}
