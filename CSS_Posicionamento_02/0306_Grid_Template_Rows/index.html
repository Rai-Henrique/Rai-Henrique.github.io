grid-template-rows

  • grid-template-rows
      Define o tamanho das linhas.
    • grid-template-rows: 100px auto.;
  • grid-auto-rows:
    • Linhas são adicionadas automaticamente com o valor de auto.
  • grid-row: 2
    • Define a linha do item, funciona da mesma forma que o grid-column.
  • grid-row: 1 / -1;
    • que vá da linha 1 até a última linha
  • grid-area
    • é a combinação do grid-row e grid-column

ativando no elemento com class"info" o display: grid; para os elementos inline(R$49 e Comprar) virar inline block.
