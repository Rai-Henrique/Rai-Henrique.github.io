z-index

Define a posição no eixo z (profundidade) dos elementos que foram posicionados fora do fluxo (relative, fixed, absolute).

no css, o elemento que está embaixo(no código) é que fica por cima do elemento da linha anterior no site, ou seja, é como se fosse uma escada onde o primeiro degrau de baixo para cima seria o Elemento 5.

É o z-index que vai definir se o elemento vai ficar por cima de todos ou por baixo de todos, além disso deve haver uma definição de Position: ;
