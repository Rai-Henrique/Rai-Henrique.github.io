Fluxo do Layout

O fluxo do layout no HTML ocorre conforme o modo de escrita definido. Por padrão, de cima para baixo da esquerda para a direita.

span

É uma tag genérica que não possui nenhum estilo pré-definido/semântica. É equivalente a uma div, mas sem o display block.

    Ex:
texto direto no body texto direto no link texto no span
texto direto na div
texto direto no body 2 texto no link 2 texto no span 2
