Editor Plugins.

  • Live Server
    • https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

      • Utilizado para atualizar automaticamente a página ao inserirmos uma nova mudança no arquivo.


  • Prettier
    • https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode
      • Serve para formatar automaticamente o documento pelo próprio Visual Studio Code.
