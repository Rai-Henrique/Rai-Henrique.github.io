Imagem svg e png


Por que usar img png ou svg? porque possuem o fundo transparente e cria um arquivo menor.

Ao aumentarmos em px o tamanho das duas imagens, será notável que a img svg(img de cima) ficará mais limpa em comparação a img png(img debaixo).

Ferramenta para facilitar o uso de imagens:

Site: https://unsplash.com/

Como usar o unsplash: https://www.youtube.com/watch?v=06wubUdlXow

Primeiro passos básicos:

  1. Entrar no site.
  2. Pesquisar por um nome específico.
  3. arrastando o cursor na imagem aparecerá a caixa para baixar.
  4. Após isso, para otimizar a imagem(deixá-la em kb), ir para o squoosh.

Ferramenta para otimizar imagens:

Site: https://squoosh.app/

Como usar o squoosh: https://www.youtube.com/watch?v=j1ROk7Yfx2U

Primeiro passos básicos:

  1. Entrar no site.
  2. Clicar no "+" para adicionar uma imagem baixada.
  3. selecionar a imagem.
  4. Após isso, para otimizar a imagem(deixá-la em kb), no site aparecerá as ferramentas para otimizar a imagem selecionada.
