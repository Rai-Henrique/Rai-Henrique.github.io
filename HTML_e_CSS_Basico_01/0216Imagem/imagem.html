Início.

Img

A Tag < img> é utilizado para adicionarmos imagens à nossa página.

    SRC
  • Atributo que define o caminho da imagem.
    ALT
  • Texto alternativo que será lido por leitores de tela(acessibilidade), robôs e que irá aparecer caso a imagem não carregue.
    MAX-WIDTH: 100%;
  • Geralmente definimos um width máximo para a imagem de 100%, assim ela não cresce além do elemento pai.

Por padrão as imagens vêm definidas como:

Display: inline-block;, por isso esse pequeno espaçamento entre elas como mostrado abaixo:

Mudando para Display: block;, com isso as imagens ficarão grudadas uma da outra.

    Iportante, o Display: Block; deve ser definido nas imagens e não no elemento pai que está segurando elas(div) já que é para juntar as imagens e não divs.

Resolução

Por que usar max-width: 100%; no seletor img ao invés de width: 100%;? porque se eu usar width: 100%; até a imagem com resulção baixa irá ficar do tamanho total do elemento pai toda borrada, já a imagem com resolução alta não terá alterações como sendo mostrado abaixo:

Imagem com resolução alta

img{
max-width: 100%;
}

Imagem com resolução baixa

img{
width: 100%;
}
